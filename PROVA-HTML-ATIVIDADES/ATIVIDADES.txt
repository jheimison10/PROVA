1- Qual tag HTML para linkar o CSS?
R= <link rel="stylesheet" href="style.css"> 
Essa linha de código é a forma padrão e mais recomendada para incluir CSS. 

2- Demonstre e explique o evento houver?
R=
 
Ex: Em HTML
<head>
    <meta charset="UTF-8">
    <title>Demonstração Hover</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <a href="#" class="meu-botao">Passe o Mouse Aqui</a>
</body>

Ex: Em CSS

.meu-botao {
    /* Estilos Padrão */
    display: inline-block;
    padding: 10px 20px;
    background-color: #007bff; /* Azul */
    color: white;
    text-decoration: none;
    border-radius: 5px;
    transition: background-color 0.3s ease, color 0.3s ease; /* Adiciona uma transição suave */
}

.meu-botao:hover {
    /* Estilos Aplicados ao Passar o Mouse */
    background-color: #0056b3; /* Azul mais escuro */
    color: #ffc107; /* Amarelo */
    cursor: pointer; /* Muda o cursor para indicar que é clicável */
}

Implementação com CSS: Pseudo-classe :hover
A maneira mais comum, eficiente e recomendada para aplicar estilos em um elemento quando o cursor do mouse passa sobre ele é usando a pseudo-classe :hover no CSS


3- Centralize 2 divs no meio da tela?
R= Para centralizar duas divs horizontal e verticalmente no meio da tela (ou dentro de um container pai), a maneira mais moderna e flexível é usar Flexbox ou Grid no CSS.
A abordagem mais comum é aninhar as duas divs em um container e aplicar Flexbox a ele.

Ex: Em HTML
<div class="container">
  <div class="box">Div 1</div>
  <div class="box">Div 2</div>
</div>

Ex:Em CSS
.container {
  /* 1. Torna o elemento um container flexível */
  display: flex;

  /* 2. Centraliza os itens **horizontalmente** */
  justify-content: center;

  /* 3. Centraliza os itens **verticalmente** */
  align-items: center;

  /*
   * 4. Define o tamanho do container.
   * Para centralizar no meio da tela (viewport),
   * o container deve cobrir a tela inteira.
   */
  height: 100vh;
  width: 100vw;

  /* Estilos opcionais para visualizar */
  border: 2px solid blue;
  gap: 20px; /* Adiciona espaçamento entre as duas divs */
}

.box {
  padding: 20px;
  background-color: lightgray;
  border: 1px solid black;
}
